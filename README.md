# **League of Legends AutoPicker(미완성)**
## **설명** 
- 리그오브레전드 픽을 자동으로 잡아주는 스크립트입니다.
- 1980x1080 해상도에서 정상적으로 작동합니다.
## **사용한 라이브러리**
- pyautogui
- pyinstaller
## **사용법**
1. main.py를 실행한다.
2. 포지션과 챔피언을 입력한다.
2. 롤 클라이언트를 실행한다.
3. 게임을한다.
